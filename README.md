## Installation

```sh
virtualenv _env
source _env/bin/activate
pip install pytest
pip install requests
sudo apt-get install jq curl -s https... | jq
```

When you are done, you can deactivate your Python environment using `deactivate`.

if you want to use jq- curl -s https ... | jq
